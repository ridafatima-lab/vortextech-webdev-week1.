# Personal Portfolio Website

## About the Project

This is my Week 1 Web Development Internship task for VortexTech.

I built a responsive single-page personal portfolio using only HTML and CSS. The website includes:

- Header with profile image and introduction
- About Me section
- Skills section
- Contact section
- Responsive design using CSS Flexbox and Media Queries

## Technologies Used

- HTML5
- CSS3
- Flexbox
- Responsive Design

## How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Double-click `index.html` or open it using Live Server in Visual Studio Code.
