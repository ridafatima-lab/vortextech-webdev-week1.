# Rida Fatima – Personal Portfolio

## Overview
This project is a responsive personal portfolio website built using HTML and CSS as part of the VortexTech Internship.

The portfolio introduces me as a Law student, Web Developer, UI/UX Designer, and Graphic Designer. It follows a legal-inspired theme with a clean and professional layout.

## Features
- Responsive design for desktop and mobile devices
- Sticky navigation bar
- Hero section with profile image
- About section
- Skills and areas of practice
- Project timeline (Docket)
- Contact section with social media links
- Modern typography and custom color palette
- Smooth scrolling and hover animations

## Technologies Used
- HTML5
- CSS3
- Google Fonts
- Flexbox
- CSS Grid

## Project Structure
```
project-folder/
│
├── index.html
├── style.css
├── profile.jpg
└── README.md
```

## How to Run
1. Download or clone this repository.
2. Open the project folder.
3. Double-click `index.html` or open it in any modern web browser.

No installation or additional software is required.
